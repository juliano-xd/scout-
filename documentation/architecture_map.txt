[ MOTORES DE ANALISE (ENGINES) ]

- Content Search: Busca textual/regex de alta performance.
  Files: include/engines/content_search_engine.hpp, src/engines/content_search_engine.cpp

- Class Search: Localizacao de classes via Dalvik/String.
  Files: include/engines/class_search_engine.hpp, src/engines/class_search_engine.cpp

- XRef Engine: Analise de referencias cruzadas e grafos de chamada.
  Files: include/engines/xref_search_engine.hpp, src/engines/xref_search_engine.cpp

- Variable Tracker: Taint Analysis (Rastreio de dados Source-to-Sink).
  Files: include/engines/variable_tracker_engine.hpp, src/engines/variable_tracker_engine.cpp

- CFG Engine: Geracao de grafos de controle de fluxo de metodos.
  Files: include/engines/cfg_engine.hpp, src/engines/cfg_engine.cpp

- UI Mapper: Correlacao de Resource IDs com bytecode.
  Files: include/engines/ui_mapper_engine.hpp, src/engines/ui_mapper_engine.cpp

- ResMap Engine: Mapeamento de nomes de recursos (public.xml).
  Files: include/engines/resource_mapping_engine.hpp, src/engines/resource_mapping_engine.cpp

- Deobfuscation: Deteccao de Base64 e analise de entropia.
  Files: include/engines/deobf_engine.hpp, src/engines/deobf_engine.cpp

- Manifest Engine: Auditoria de AndroidManifest.xml.
  Files: include/engines/manifest_engine.hpp, src/engines/manifest_engine.cpp

- Class Inspector: Extracao de DNA e metadados de classe.
  Files: include/engines/class_inspector_engine.hpp, src/engines/class_inspector_engine.cpp

- Dominator Analyzer: Calculo de IPD para dissipacao de fluxo.
  Files: include/engines/dominator_analyzer.hpp, src/engines/dominator_analyzer.cpp


[ NUCLEO E INFRAESTRUTURA (CORE) ]

- Main Loop: Orquestracao de CLI e Despacho de Motores.
  File: src/main.cpp

- CLI Parser: Sistema de flags e configuracao (ScoutConfig).
  File: include/cli/parser.hpp

- Scanner Core: Motor de varredura e filtragem de diretorios.
  File: include/core/scanner.hpp

- Analysis Context: Estado global da analise e diretorio raiz.
  File: include/core/analysis_context.hpp

- S-Expr Formatter: Protocolo de comunicacao AI-Native.
  File: src/formatters/sexpr_formatter.cpp


[ BIBLIOTECAS UTILITARIAS (UTILS) ]

- sexpr.hpp: Logica de construcao de S-Expressions.
- mmap_file.hpp: Performance extrema em leitura de arquivos gigantes.
- string_utils.hpp: Helpers de texto, Base64 e entropia.
- filesystem.hpp: Abstracao de manipulacao de arquivos do OS.
